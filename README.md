# Bienvenido a Inventory-PTR

# Objetivo General


Desarrollar un software de inventario y ventas  para **Piñaterias** ,que permite gestionar productos,controlar stock en tiempo real,registar ventas de forma rápida y generar reportes de ingresos.
El sistema facilita la organización del negocio, evita pérdidas por falta de inventario y mejora la atención al cliente con un punto de venta más sencillo.

# Objetivos Específicos

1. Diseñar un módulo de CRUD para usuarios.
2. Diseñar un módulo de CRUD para productos disponibles en la piñateria.
3. Implementar un sistema de punto de venta que facilite e registro rapido de ventas diarias.
4. Desarrollar reportes de ingresos,productos y movimientos de inventario.
5. Pemitir el control de stock mínimo para evitar productos faltantes. 
6. Crear una interfaz sencilla que facilite el uso del sistema.
7. Garantizar la seguridad de la información.
